# 코딩 테스트 알고리즘 및 메서드 모음

코딩 테스트에 자주 나오는 알고리즘과 메서드들의 사용 예제를 모아놓은 레포지토리입니다.

## 📁 구조

- **Array/List** - 배열 및 리스트 관련 알고리즘
- **String** - 문자열 처리 관련
- **Math** - 수학 관련 알고리즘
- **Graph** - 그래프 탐색 (BFS, DFS, 다익스트라 등)
- **DP** - 동적 프로그래밍
- **Sorting** - 정렬 알고리즘
- **Searching** - 이진 탐색 등

## 🎯 주요 내용

각 파일에는 다음이 포함됩니다:
- 알고리즘/메서드 설명
- 코드 구현
- 사용 예제
- 시간/공간 복잡도

## 📝 사용 방법

해당 카테고리의 파일을 참고하여 학습하고 필요할 때 참고하세요.

## 🧪 테스트

JUnit 5를 사용한 포괄적인 단위 테스트가 포함되어 있습니다.

### 테스트 클래스
- **ArrayTest** - 배열 알고리즘 테스트 (교집합, 회전, 합치기 등) - 13개 테스트 케이스
- **StringTest** - 문자열 알고리즘 테스트 (팰린드롬, 아나그램 등) - 19개 테스트 케이스
- **MathTest** - 수학 알고리즘 테스트 (GCD, 소수, 팩토리얼 등) - 19개 테스트 케이스
- **GraphTest** - 그래프 알고리즘 테스트 (BFS, DFS, 사이클 감지 등) - 19개 테스트 케이스

### 테스트 실행

```bash
# Maven을 사용한 테스트 실행
mvn test

# 특정 테스트 클래스만 실행
mvn test -Dtest=ArrayTest

# 특정 테스트 메서드만 실행
mvn test -Dtest=ArrayTest#testIntersection
```

### 테스트 통계
- **총 테스트 케이스**: 70+
- **커버리지**: 주요 알고리즘 함수 대부분 포함
- **테스트 유형**: 정상 케이스, 엣지 케이스, 경계값 테스트 등

## 🔧 빌드

```bash
# 프로젝트 빌드
mvn clean install

# JAR 파일 생성
mvn clean package
```

## 📦 의존성
